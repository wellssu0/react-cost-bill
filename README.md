## 每日记账

### 介绍：

- react框架实现的简单的记账单页应用
- 外观用了bootstrap4的Grid布局、Form组件、List组件，recharts的饼状图、react-ionicons的图标
- 主要技术栈react、react-router、redux、redux-saga、react-redux、axios、json-server等

### 功能：

- 实现异步添加、修改、删除记账记录
- 实现页面的路由跳转
- 实现数据的可视化
